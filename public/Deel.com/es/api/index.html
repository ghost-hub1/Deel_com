Preguntas frecuentes

¿Cómo empiezo con Deel API?

Visita developer.deel.com para explorar los documentos, las guías y la página de pruebas en vivo de la API de Deel. A continuación, crea un entorno aislado dentro de la plataforma Deel para empezar a crear integraciones con datos de muestra. Finalmente, genera tokens de API en vivo en Developer Center cuando estés listo para implementar tu integración a producción.

¿Cuánto tiempo lleva configurar una API?

Esto varía según el sistema que quieras sincronizar con la plataforma Deel, qué recursos tienes disponibles y la gama de funcionalidades que deseas configurar dentro de la API. Configurar una prueba de concepto básica puede tardar tan solo un día o unas semanas según la priorización interna de la implementación.

¿Se puede integrar Deel API con mi stack de software actual?

Por supuesto. Deel API está diseñada para funcionar bien con el software que ya estás utilizando. Tanto si tienes un sistema interno como si dependes de aplicaciones de terceros para RR. HH., finanzas u operaciones, nuestra API puede integrarse para ayudarte a optimizar tus procesos. Hemos trabajado arduamente para asegurarnos de que sea compatible con una amplia gama de plataformas, y nuestro equipo de asistencia técnica está a tu disposición para ayudarte a integrarlo lo mejor posible.

¿Qué tipo de soporte ofrece Deel para la implementación de API?

Sabemos que empezar con una nueva API puede conllevar sus propios desafíos; por eso ofrecemos soporte dedicado durante todo el proceso de implementación. Damos documentos detallados, y también tenemos una comunidad de desarrolladores donde puedes intercambiar ideas y soluciones. Nuestro objetivo es asegurarnos de que cuentes con toda la ayuda que necesitas para sacar el máximo provecho de Deel API.
